# pitt_pbft

This project use `clap`to control command line.

Get help

`cargo run -- -h`

```

USAGE:
    server [OPTIONS]

OPTIONS:
    -h, --help           Print help information
    -p, --port <PORT>    bind port, default port 2022 [default: 2022]
    -V, --version        Print version information
```



Specify port, default is 2022

`cargo run -- -p 2022`
